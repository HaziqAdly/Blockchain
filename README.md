# Blockchain Project: Bank Negara Cryptocurrency Launderer
## Installation
- Install Truffle
<pre>npm install -g truffle </pre>
- Run development console
<pre>truffle develop </pre>
- Compile and migrate the contract
<pre>compile
migrate </pre>
- Test smart contract
<pre>test</pre>
